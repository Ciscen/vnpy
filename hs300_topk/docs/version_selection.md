# HS300 Top-K 版本评估与选型文档 (2026.05)

在经历了 2022-2023 的深度熊市和 2024 之后的牛市周期回测后，我们对多个策略版本进行了严格的样本外验证与全周期压测。

根据回测数据（包含牛熊市全周期：2022.01 - 2026.04），现对后续迭代用到的核心版本进行定调。

## 一、择优保留的两个核心版本

### 1. V1.4+Lag-3 (激进/纯多头基线)
- **标签设计**：`friday_close`（保守版绝对收益标签，周五收盘相对周二开盘上涨3%）
- **核心特征**：引入前3天全量因子（Lag-3，632维特征），集中持仓 Top-5。
- **全周期表现**：
  - 牛市阶段（如2024-2026）具有极强的爆发力和高弹性（OOS阶段表现最佳）。
  - 熊市阶段（如2022-2023）回撤巨大（达 -51%），受系统性风险影响严重。
- **保留理由**：
  - 作为**牛市高弹性的标杆（Benchmark）**。它证明了基于树模型的量价动量特征在上升趋势中的绝对获利能力。
  - 后续如果要测试做空对冲工具（如股指期货空头），该版本可直接作为优秀的纯多头底仓（Long-Leg）。

### 2. V1.5 Ensemble (稳健/全天候生产基线)
- **标签设计**：双模型融合（`high_touch` Beta模型 + `excess_return` Alpha模型）。
- **核心机制**：通过基准（沪深300）的 MA60 均线作为宏观状态判别器（Regime Switch）。大盘在 MA60 之上时使用高弹性 Beta 模型，在 MA60 之下时平滑切换为追求超额收益的 Alpha 模型。
- **全周期表现**：
  - **总收益率**：95.07%
  - **年化收益率**：21.81%
  - **最大回撤**：-35.62%（相比 V1.4 熊市阶段的 -51% 有了质的改善，明显跑赢基准）
  - **Sharpe**：0.70
- **保留理由**：
  - **完美平衡了防守与进攻**。解决了单一树模型在训练时面对“牛市多暴涨样本、熊市全阴跌样本”导致的分布混淆（混合分布导致目标冲突）。
  - **实盘生产的首选**。通过物理隔离两个模型再行组合，既保住了熊市的相对 Alpha（超额），又没有牺牲太多牛市的绝对收益。

## 二、被废弃的版本及原因

### 1. V1.3 及之前的低集中度版本 (Top-10)
- **废弃理由**：持仓过度分散（Top-10）导致收益平庸，手续费占比高。在引入动态 K（Dynamic K）和严格止损后，Top-5 甚至 Top-4 的集中持仓更能发挥 XGBoost 的头部预测优势。

### 2. V1.4R (纯硬性组合回撤熔断)
- **废弃理由**：简单粗暴地在组合回撤达到 15% 时清仓并冷却。
- **陷阱**：在震荡市或牛市初期，很容易被短暂的回调触发熔断，导致完美踏空后续的主升浪（踏空风险极大）。这种后验的“一刀切”风控对长期复利伤害甚至大于短期的浮亏。

### 3. Dynamic Regime Label (单一模型动态标签)
- **原理**：在 `labeler.py` 中，牛市打高弹性标签，熊市打超额收益标签，混合后输入给同一个 XGBoost 模型训练。
- **废弃理由**：**模型困惑与分布冲突**。同一个树模型在学习过程中，面对相似的量价形态，由于人为在时间轴上强行切换了 Target 目标，导致叶子节点的纯度极低（信息增益难以收敛）。最终表现甚至不如单纯的绝对收益模型。
- **教训**：不要用单一模型去拟合包含结构性断裂（Structural Break）的多重逻辑。**混合宏观逻辑应该在模型外部（Ensemble/Meta-Model）解决，而不是在 Label 内部解决。**
