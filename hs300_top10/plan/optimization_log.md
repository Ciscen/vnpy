### 方向 2: 标签设计优化

**现状**: 二分类标签 (5% 阈值)，正例率 24.47%。

**优化思路**:

| 子方向 | 描述 | 预期效果 | 优先级 |
|---|---|---|---|
| 2a. 自适应阈值 | 根据近 N 期波动率动态调整阈值 | 低波时降阈值，高波时升 | 高 |
| 2b. 多分类标签 | 3 类：涨 5%+, 跌 3%+, 中性 | 区分"不买"和"做空" | 中 |
| 2c. 回归目标 | 预测周收益率，再按排名选股 | 更细粒度排序 | 中 |
| 2d. 信息比率标签 | label = (收益 - 全市场中位数) / 截面波动 | 去除市场系统性波动 | 低 |

**实施方案 (2a 优先)**:
```python
# 动态阈值: 使用近 20 周的平均周振幅
weekly_amplitude = df.group_by("week").agg(
    (pl.col("high").max() - pl.col("low").min()) / pl.col("open").first()
)
threshold = weekly_amplitude.rolling_mean(20) * factor
```

### 方向 3: 模型升级

**现状**: 单一 XGBoost，default 超参。

**优化思路**:

| 子方向 | 描述 | 预期效果 | 优先级 |
|---|---|---|---|
| 3a. 超参优化 | Optuna 搜索 (depth, lr, subsample, colsample) | Sharpe 提升 0.1~0.3 | 高 |
| 3b. LightGBM 替代 | GOSS 采样 + 直方图加速 | 训练快 2-3x，精度相当 | 高 |
| 3c. 模型融合 | XGB + LGBM + CatBoost 加权平均 | 降低单模型方差 | 中 |
| 3d. 样本不平衡 | SMOTE / scale_pos_weight 调整 | 提升正例召回率 | 中 |
| 3e. 时序 CV | TimeSeriesSplit 5-fold 替代单次 80/20 | 更稳健的验证 | 中 |
| 3f. 深度模型 | TabNet / Temporal Fusion Transformer | 捕获非线性时序依赖 | 低 |

**实施方案 (3a + 3b 优先)**:
```python
import optuna

def objective(trial):
    params = {
        "max_depth": trial.suggest_int("max_depth", 3, 10),
        "learning_rate": trial.suggest_float("learning_rate", 0.01, 0.3, log=True),
        "subsample": trial.suggest_float("subsample", 0.5, 1.0),
        "colsample_bytree": trial.suggest_float("colsample_bytree", 0.3, 1.0),
        "min_child_weight": trial.suggest_int("min_child_weight", 1, 10),
        "reg_alpha": trial.suggest_float("reg_alpha", 1e-8, 10.0, log=True),
        "reg_lambda": trial.suggest_float("reg_lambda", 1e-8, 10.0, log=True),
    }
    # 时序交叉验证
    scores = time_series_cv(params, train_df, n_splits=5)
    return np.mean(scores)
```

### 方向 4: 选股与组合优化

**现状**: 按概率 Top-10 等权买入。

**优化思路**:

| 子方向 | 描述 | 预期效果 | 优先级 |
|---|---|---|---|
| 4a. 概率加权 | 按预测概率分配仓位权重 | 高信心股票仓位更重 | 高 |
| 4b. 动态 K 值 | 模型信心不足时降低持仓数量 | 低信号期减少交易 | 高 |
| 4c. 去相关过滤 | 同行业最多选 N 只 | 降低行业集中风险 | 中 |
| 4d. 流动性过滤 | 剔除日均成交额 < 阈值的股票 | 避免冲击成本 | 中 |
| 4e. 均值-方差优化 | Markowitz 优化权重分配 | 理论最优风险收益比 | 低 |

**实施方案 (4a + 4b 优先)**:
```python
# 概率加权分配
top_k_df = signal_df.head(top_k)
weights = top_k_df["signal"] / top_k_df["signal"].sum()
for sym, w in zip(top_k_df["vt_symbol"], weights):
    buy_value = total_cash * cash_ratio * w

# 动态 K: 当最高概率 < 阈值时缩减持仓
max_prob = signal_df["signal"].max()
if max_prob < 0.3:
    effective_k = max(3, top_k // 2)
```

## 附录: 评估标准

策略优化的核心评估指标及优先级：

1. **Sharpe Ratio** ≥ 1.5（核心目标）
2. **最大回撤** ≤ -10%
3. **收益回撤比** ≥ 3.0
4. **年化收益率** ≥ 25%
5. **换手率** 合理（月度 < 200%）

优化的最终目标是在风险可控的前提下最大化风险调整后收益（Sharpe），而非单纯追求高收益。
